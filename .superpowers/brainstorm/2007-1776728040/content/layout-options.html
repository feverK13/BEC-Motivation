Варіанти макету навігації

Який підхід краще підходить для мотиваційного листа?

A

Фіксована навігація зверху

Класичний односторінковий дизайн. Навігація завжди видима, кожна секція займає viewport. Чисто, зрозуміло, професійно.

B

Бічна навігація

Навігація зліва на десктопі, згортається вгорі на мобільних. Більше місця для контенту, сучасний вигляд.

C

Плаваюча навігація

Навігація приховується при скролі вниз, з'являється при скролі вгору. Максимум місця для контенту, мінімалістично.
