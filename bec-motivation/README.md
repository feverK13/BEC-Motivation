# BEC Motivation Website

Персональний сайт мотиваційного листа для подачі на роль у BEST Engineering Competition.

## Стек технологій

- React 19
- TypeScript
- Vite
- CSS Modules

## Інструкція із запуску

```bash
npm install
npm run dev
```

Відкрий http://localhost:5173 у браузері.

## Production build

```bash
npm run build
```

Збірка буде створена у папці `dist/`.
